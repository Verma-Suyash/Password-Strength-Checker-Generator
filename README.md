# Password-Strength-Checker-Generator
Password Security Pro is a web-based app built with HTML, CSS, and JavaScript that checks password strength in real time and generates secure passwords. It analyzes complexity using multiple criteria, provides instant feedback, and includes a customizable generator with a clean, responsive UI.

🔐 Password Security Pro

A web-based Password Strength Checker & Generator built using HTML, CSS, and JavaScript. This project helps users evaluate password strength in real time and generate secure passwords based on customizable criteria.

🚀 Features

🔍 Password Strength Checker
  Real-time analysis of password strength
  Checks length, uppercase, lowercase, numbers, and symbols
  Displays strength using percentage and labels
  
🔑 Password Generator
  Generate strong, random passwords
  Customizable length and character types
  One-click copy to clipboard
  
🎨 User Interface
  Clean and responsive design
  Dark/Light theme toggle
  Smooth animations and dynamic effects
  
🛠️ Tech Stack
  Frontend: HTML, CSS
  Logic: JavaScript
  Libraries: Bootstrap, Font Awesome, GSAP, Lottie
  
📂 Project Structure

  This project aims to promote better cybersecurity practices by helping users create strong and secure passwords while demonstrating practical web development     skills.
